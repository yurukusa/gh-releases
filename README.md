# gh-releases

Browse GitHub release notes for any repository. Markdown rendering, version filter, and asset downloads.

**[Live Demo →](https://yurukusa.github.io/gh-releases/)**

## Features
- View release notes with rendered Markdown
- Filter and search across versions
- Download release assets directly
- No authentication required (uses public GitHub API)

## Usage
Enter a GitHub repository (e.g. `facebook/react`) to browse its release history and changelogs.

## Tech
- Single HTML file, vanilla JavaScript
- No dependencies, no build step
- Works offline after first load
